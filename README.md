# ビンゴゲーム

Javaで制作したビンゴゲームです。

## ■ 仕様
問題文の指示と『出力例.txt』に沿い制作。
### 設計の意図
  
- **拡張性の考慮**： 5x5以外のカードサイズや、終了条件（ビンゴ数）の変化にも対応可能にしています。
- **役割によるクラス分け**：各クラスに役割に応じた処理を実装し、全体の構成を見やすくしています。
- **操作方法の提示**：ゲームの開始、進行、終了といったタイミングを視覚的にわかるようにしています。

## ■ 実行方法
1. Main.java を実行。
2. Enterキーを押すと抽選が始まります。(終了条件達成までループ)
3. ループ終了後、yを押すと新たにゲームを開始、nを押すとゲームを終了します。

## ■ 開発環境
- Java 21 
- eclipse
